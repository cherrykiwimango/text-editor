# Text Editor

A text editor built from scratch in C, following Antirez's Kilo editor tutorial.

## Installation

```bash
git clone git@github.com/cherrykiwimango/text-editor.git
cd text-editor
sudo make install
```

After installation, you can optionally remove the source directory:

```bash
cd ..
rm -rf text-editor
```

Run the editor with:

```bash
kilo
```
